# 📚 일산 직업학교 LMS & 구직 매칭 시스템
> **Flutter와 Firebase를 활용한 교육 관리 및 취업 지원 원스톱 플랫폼**
> 권한별 맞춤 인터페이스와 강력한 보안 라우팅을 제공합니다.

---

## 🛠 Tech Stack

| 구분 | 기술 스택 | 상세 내용 |
| :--- | :--- | :--- |
| **Frontend** | **Flutter** | Web & Mobile 응답형 UI, 효율적인 상태 관리 |
| **Backend** | **Firebase** | Firestore(DB), Auth(인증), Storage(파일 관리) |
| **Serverless** | **Cloud Functions** | FCM 푸시 알림 트리거 및 자동화 로직 |
| **Automation** | **Pub/Sub Scheduler** | 매일 자정 종료 강좌 자동 상태 변경 (Cron Job) |

## 🏗 핵심 아키텍처 및 보안 정책 (Core Architecture)

### 1. 보안 및 권한 정책
* **Enum-Based Role:** 한글이 아닌 `SUPER_ADMIN`, `INSTRUCTOR`, `STUDENT` 영문 코드로 모든 권한을 엄격히 통제합니다.
* **Route Guard:** 권한 없는 페이지나 세션 만료 시 즉시 접근을 차단하고 로그인 페이지로 강제 리다이렉트합니다.
* **Soft & Hard Delete:** 데이터 무결성을 위해 DB 레코드는 상태값 변경(Soft)으로 이력을 보존하며, 서버 파일(이미지 등)은 물리적 삭제(Hard)를 적용해 저장 공간을 최적화합니다.

### 2. 스마트 데이터 관리(차후 개발)
* **Clean Server:** 게시글 수정/삭제 시 파이어베이스 스토리지 내의 기존 파일을 자동 삭제하여 찌꺼기 데이터를 방지합니다.
* **용량 제한 유효성 검사:** 웹 환경 크래시 방지를 위해 `cross_file` 기반으로 모든 첨부파일을 **3MB 이하**로 제한합니다.

## 👥 사용자 권한별 주요 기능 (User Roles)

### 👑 최고 관리자 (Super Admin)
> *최초 계정은 개발자가 직접 생성하며, 전체 시스템의 제어권을 가집니다.*
* **통계 대시보드:** 활성 교사, 진행 중인 강좌, 전체 학생 수를 실시간으로 모니터링합니다.
* **강좌 라이프사이클:** 스마트 에디터로 강좌를 개설하며, 종료일 기준 자정 스케줄러를 통해 자동 마감 처리합니다.
* **지능형 교사 삭제 방어:** 담당 강좌나 대기 학생이 있는 교사 삭제 시 차단 메시지를 띄우며, **[인수인계 팝업]**을 통해 담당 권한을 타 교사에게 트랜잭션으로 안전하게 이관합니다.

### 👨‍🏫 교사 (Instructor)
> *담당 학급 운영과 학생들의 취업 지원을 담당하는 핵심 운영진입니다.*
* **스마트 대시보드:** 담당 반 카드, 승인 대기 리스트, 구직 현황을 탭(Tab) UI로 관리합니다.
* **학생 관리 기능:** 가입 신청 승인/거절 및 비밀번호 분실 시 **임시 비밀번호 발급(원클릭 클립보드 복사)** 기능을 제공합니다.
* **구직 공고 매칭:** 특정 반을 타겟팅하여 채용 공고를 등록하고, 지원한 학생들의 승인 및 취소 상태를 실시간으로 업데이트합니다.

### 🎓 학생 (Student)
> *학습 및 구직 활동을 수행하며 계정 상태(Status)에 따라 화면 접근 권한이 자동 조절됩니다.*(차후 오픈)
* **상태별 맞춤 라우팅:** `승인대기(pending)`, `수강중(active)`, `수료/퇴소(graduated/dropped)` 상태에 따라 대시보드 혹은 재신청 화면으로 자동 안내합니다.
* **스마트 구직 리스트:** 15개 단위 무한 스크롤(Infinite Scroll)로 채용 정보를 탐색하고 상세 페이지에서 즉시 지원합니다.
* **계층형 Q&A:** 구직 공고 내에서 교사와 대댓글로 소통하며, 이메일 마스킹 처리로 개인정보를 보호합니다.

## 🔔 알림 및 통신 시스템 (Notice & FCM)
* **타겟팅 푸시 알림:** 전체 공지뿐만 아니라 교사가 **특정 반 학생들만 지정**하여 실시간 푸시를 발송할 수 있습니다.(차후 업데이트)
* **Cloud Functions 연동:** 공지 등록 시 백엔드에서 트리거가 작동하여 대상자의 FCM 토큰을 수집, 알림을 자동 전송합니다.
* **댓글 카운팅 로직:** 삭제된 댓글을 제외한 실제 활성 댓글과 대댓글의 합계를 실시간으로 계산하여 노출합니다.
